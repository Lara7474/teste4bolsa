# Modelos DOCX ajustados para o Lovable

Ajustes aplicados:
- Placeholders padronizados em snake_case, sem acentos, sem espacos e sem caracteres especiais.
- Aliases do Termo de Referencia convertidos para os mesmos nomes usados no sistema.
- Removidos textos residuais de exemplos próximos a placeholders em alguns modelos.
- Mantida a estrutura DOCX original para preservar a formatacao.
